npm简介

npm，全称为node package manager，是Node的模块管理器，也可以说是JavaScript的包管理工具。npm体系有npm工具客户端和npm包仓库两部分。npm包官方仓库为https://www.npmjs.com，使用npm客户端工具可以在本地安装、删除JavaScript包，功能非常强大。
